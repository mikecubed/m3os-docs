## Implementation outline

- Parse framebuffer information from boot data and expose a simple drawing API.
- Add text rendering with a fixed bitmap font.
- Extend the console service to own the visible terminal state.
- Build a tiny shell that speaks only to documented services.
- Keep the command model simple and text-oriented.
