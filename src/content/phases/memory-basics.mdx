## Implementation outline

- Store a `'static` reference to the bootloader memory map (the regions slice is valid for the kernel's lifetime) and parse it into typed kernel structures.
- Implement a simple frame allocator with conservative rules.
- Expose page-table operations through safe wrappers around `x86_64`.
- Map a small heap and initialize the allocator.
- Add simple allocation tests and logging for confidence.
