## Why this phase matters

Claude Code runs natively inside m3OS, using the supported Node, network, git, and GitHub tooling to read code, run commands, and participate in the same developer workflow the earlier post-1.0 phases made possible.

## How it works in m3OS

The roadmap entry defines the learning scope for this phase; a deeper implementation doc has not been matched yet.

## How it builds on earlier phases

This phase builds on Node.js and turns that foundation into Claude Code.

## Implementation outline

- Define the supported installation and launch path for the tool.
- Validate the file, shell, git, and network workflows the tool depends on.
- Document the credential-handling and support-boundary story.
- Test the supported workflows inside m3OS end-to-end.
- Update the standalone roadmap and top-level docs for the new milestone.
