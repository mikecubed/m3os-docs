## Why this phase matters

m3OS can make HTTPS connections to the outside world. The GitHub CLI (`gh`) runs natively, enabling pull requests, issues, and CI status checks from the command line. git gains HTTPS remote support for clone, push, and pull to GitHub. DNS resolution works via musl's built-in resolver.
