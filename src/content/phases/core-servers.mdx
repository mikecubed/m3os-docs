## Implementation outline

- Decide what the kernel launches directly and what `init` launches afterward.
- Build the service registration model used for early discovery.
- Move serial console behavior behind a console server.
- Route keyboard notifications into a dedicated keyboard server.
- Keep restart and crash handling simple but visible.
