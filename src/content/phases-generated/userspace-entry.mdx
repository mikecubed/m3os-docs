## How it builds on earlier phases

## Implementation outline

- Create a process abstraction separate from kernel threads.
- Build a minimal userspace address space with code and stack mappings.
- Enter ring 3 through a carefully documented transition path.
- Install the syscall entry point and argument convention.
- Add one tiny userspace binary to exercise the path.
