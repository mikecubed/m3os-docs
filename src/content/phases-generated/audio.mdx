## Why this phase matters

The OS can play sound. A sound card driver (Intel HD Audio or AC'97, emulated by QEMU) outputs PCM audio to the host's speakers. Userspace programs write audio samples to a device file or via a dedicated syscall. DOOM plays sound effects and music.

## Implementation outline

- Start with PC speaker beep — validate audio output path.
- Detect HDA or AC'97 device via PCI enumeration.
- Initialize the sound card: reset, configure output stream, set up BDL.
- Implement DMA ring buffer management.
- Implement `/dev/dsp` device with `open`/`write`/`ioctl`.
- Write a test program that plays a sine wave.
- Integrate with DOOM platform layer for sound effects.
- Optionally implement basic mixing for multiple audio sources.
